# Como fazer o teste

1. Abrir o postman 
2. Selecionar o verbo GET
3. Digitar a seguinte URL "localhost:3000/"
4. Teste com o POST 
5. Selecionar o verbo POST 
6. Digiar a seguinte URL "localhost:3000/persons"
7. Clicar em 'body'
8. Seleciona o 'raw'
9. Selecionar 'json'
10. Digitar o seguinte JSON
```
{
    "Name": "Anderson"
}
```
